Logga in för verksamhetsstatistik

För att fortsätta till dom inloggade sidorna för verksamhetsstatistik måste du logga in med ditt SITHS-kort.

Detta kan bero på att du inte har rätt behörighet för att se statistik för verksamheten.

Gör följande:

Sätt SITHS-kortet i din kortläsare och klicka sedan på knappen "SITHS login"
