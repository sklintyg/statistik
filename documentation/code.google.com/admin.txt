= Vårdens intyg - statistiktjänst =

Statistiktjänsten är en tjänst som lagrar valda delar av avidentifierad läkarintygsinformation med målsättningen att kunna presentera uppföljning för inledningsvis verksamhetsansvariga på vårdgivarnivå samt för allmänheten på nationell nivå och på sikt även för andra intressenter. Informationen som sådan ägs av vårdgivaren som formellt är den som bestämmer vilka som får tillgång till rapporterna ifråga.

== Relaterade projekt ==
Statistikinformation kommer från den nationella intygstjänsten, se separat projektplats http://code.google.com/p/inera-certificate.
