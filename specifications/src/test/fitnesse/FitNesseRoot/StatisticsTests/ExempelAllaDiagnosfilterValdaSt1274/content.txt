Testet är ignorerat eftersom kommentaren nedan (att värdet ska vara sant när inget diagnosfilter är valt oavsett övriga filter). Detta stämmer inte eftersom filtren 
enbart ses som en helhet, dvs är något "delfilter" aktivt så kommer samtliga "delfilter" att tas i beaktning. I GUIt är detta dock inte något som användaren behöver
tänka på eftersom default-inställningen för diagnosfiltret är att alla diagnoser är valda.


Detta är ett exempel för att visa hur man kan testa värdet som visar om alla diagnoser är valda i filtret eller inte.
Det värdet används i guit för att avgöra om listan över valda diagnoser ska visas i rapporten (STATISTIK-1274).

!3 Givet att:

| script | dagens datum | 2014-12-06 |

| script | troskel varde | 1 |

| Enbart foljande intyg finns |
| personnr | diagnoskod | start    | slut       | enhet  |
| 19671122-2940 | G01 | 2014-04-01 | 2014-06-28 | enhet1 |


!3 Då gäller:

När inget diagnosfilter ställts in, så ska värdet på allaDiagnosfilterValda vara sant, även om andra filter än diagnosfiltret använts.

| inloggad som | vårdenhet | vårdgivarnivå | filter kapitel | filter enheter | filter verksamhetstyper | filter startdatum | filter slutdatum | allaDiagnosfilterValda? |
| user2        | enhet1    | true          | G00-G99        | enhet1,enhet2  | 02                      | 2014-06-01        | 2014-09-30       | false        |
| user2        | enhet1    | true          |                |                | 02                      | 2014-06-01        | 2014-10-30       | true         |
