Två intyg för samma patient utgör ett sjukfall. Intygen är utfärdade av olika läkare på samma enhet, men de har samma namn.

Sjukfallet räknas med en gång för varje läkare. De två läkarna presenteras med namn och HSA-id för att kunna skilja dem åt.


!3 Givet att:
| script | dagens datum | 2014-11-28 |

| script | troskel varde | 1 |

| Foljande lakare finns |
| id      | förnamn | efternamn |
| lakare1 | Anna    | Andersson |
| lakare2 | Anna    | Andersson |


| Enbart foljande intyg finns |
| personnr      | start      | slut       | enhet  | läkare  | kommentar |
| 19880701-0015 | 2014-06-01 | 2014-06-15 | enhet1 | lakare1 |           |
| 19880701-0015 | 2014-06-16 | 2014-07-30 | enhet1 | lakare2 |           |

Inloggad nivå
| Sjukfall i rapporten sjukfall per lakare |
| inloggad som | vårdgivarnivå | läkare                 | män? | kvinnor? |
| user1        | false         | Anna Andersson lakare1 | 1    | 0        |
| user1        | false         | Anna Andersson lakare2 | 1    | 0        |
